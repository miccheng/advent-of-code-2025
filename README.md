# Advent of Code 2025

My solution for [AOC2025](https://adventofcode.com/2025).

## To setup

1. Ensure you have Node 22 (or greater)

2. Install dependencies:

    ```bash
    npm install
    ```

## To Run Test

Unit tests are written in Vitest.

```
npm run test
```

## To execute the Input to get the answer

```
npm run solve
```
